# Code of Conduct

## Our Pledge

We are committed to making participation in Open-QnA a harassment-free experience for everyone.

## Our Standards

Examples of behavior that contributes to a positive environment include:
- Being respectful and constructive.
- Accepting and giving feedback gracefully.
- Prioritizing project health over personal preference disputes.

Unacceptable behavior includes harassment, personal attacks, and discriminatory language.

## Enforcement

Project maintainers are responsible for clarifying and enforcing these standards and may remove contributions that violate this Code of Conduct.

## Scope

This Code of Conduct applies within all project spaces, including issues, pull requests, and review discussions.
